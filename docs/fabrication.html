Fabrications

2Dホラーゲーム「夢」

2020年6月からUnityインターハイ2020に向けて3ヶ月の間制作していたゲーム。
何も見えない暗闇の中を主人公が懐中電灯を頼りに進んでいき、怪物が徘徊する恐ろしい建物からの脱出を試みる、というもの。
このゲームを作るきっかけになったのはある日自分が見た悪夢。
大会の結果が発表され次第、Windows版とMac版での配信を予定している。

オンラインゲーム会社説明会

2020年2月26日と3月11日に株式会社ビヨンドが行った「オンラインゲーム会社説明会」にて使用されたマップを制作、本番も裏の方でマップの操作を担当。
その後ビヨンドの広報担当さんのツイートの画像で満面の笑みでピースする自分が写っているのを発見。
これをきっかけにSNSの自分のアカウントで初めて顔を出す。

Ultimate Kettle

ニコニコがやってる「自作ゲームフェス新人賞2020」に応募するためにUnityで作った2Dシューティングゲーム。
自分が撃った弾が帰ってくるというなかなかのクソゲー。
締め切り1ヶ月前の段階でほぼやったことのないC#を勉強しながら作るというハードモードな環境の中で制作された。多分面白いはず。

Minecraft Plugin

Javaで開発した、マインクラフトの中で日本の昔からの遊び「だるまさんがころんだ」ができるようになるプラグイン。
オブジェクト指向の言語の勉強がてらいろいろやってみた。
最近パワーアップを検討中。
自分の中ではなかなかに完成度は高い方だと思っている。

HomeWars

Scratchで作った、家の中を舞台にしたシューティングゲーム。
PCNプロコン2017夏の大会にて最優秀賞を受賞。
そして自分が初めて世に出したゲームである。
多分ラスボスより途中のゴキブリの方が強い。
