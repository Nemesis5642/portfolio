Welcome to this page!!

Profile

Nemesis

2003年10月13日、この世に爆誕。現在、17才。高校2年生。
趣味の絵やプログラミングや筋トレやゲームやPokemonGOなどしながら、ゲームの大会などに作品を応募するなどしています。
発想力に関しては誰にも負けないと自負できるぐらいには自信があり、この発想力を活かし、周りとは一味違う変わった絵を描いたりデザインをしたりゲームを作ったりなどしています。
学校での二つ名は「狂人」です。人狼などで役職をCOすると真っ先に疑われそうで怖いです。
ちなみに画像は僕です。マグロのコスプレをしています。

Skills

プログラミングに関してはJava、C#、Flutter、HTML、CSS、JSなど、 おもにオブジェクト指向を中心とした言語を得意としています。
IntelliJやVScode、AndroidStudio、SublimeTextなどのエディタも扱えます。
さらに、「Minecraft」や「MagicaVoxel」による作品制作ができたり、イラストが描けたりもします。このページで使われてるイラストやらは大体自分の作品。

History

2020年

  • 2019年11月から制作していた2Dシューティングゲームが1月に完成。RPGアツマールにて「Ultimate Kettle ~究極のやかん~」という題名で公開する。
  • 2月26日,3月11日に株式会社ビヨンドが行った「オープン型オンラインゲーム会社説明会」にて使用されたワールド及び案内役のスキンを制作。
  • 6月からUnityによる2Dホラーゲームを制作開始。8月29日に完成し、「夢」というタイトルでUnityインターハイに応募。現在結果待ち。

2019年

  • 12月1日に「GDG DevFest Kobe 2019」にて発表を行う。
  • 11月からUnityで2Dシューティングゲームを制作開始。
  • 2019年4月からMinecraftのプラグイン「だるまさんがころんだ」とこのポートフォリオの制作を始め、９月に完成。

2003年~

  • 14才の時にパソコンをWindowsからMacに変える。その後、PCNプロコン2017夏の大会に出場し、2017年10月1日、最優秀賞を受賞する。
  • 12才の時に学校の帰り道でパンフレットを受け取ったのをきっかけにプログラミングスクールへ通い始める。
  • 11才の時に先生の話を聞いていなかったせいで夏休みの宿題の絵を提出する場所がどこかわからず、先生に聞けばよかったものをわからないまま提出した結果、そのまま神戸市の絵のコンクールに作品を応募し銀賞を受賞する。ついでに「人の話は聞く必要のあるものとないものがある」という教訓を得る。これをきっかけに自分に自信が持てるようになる。
  • 10才でパソコンを触り始める。人生初のパソコンはWindows。
  • 3才の時にコーラを飲み、「最高！」と叫ぶ。おそらくこの頃から絵を描き始めたことを示す絵が発見される(現在行方不明)。
  • 2003年10月13日、爆誕する。
